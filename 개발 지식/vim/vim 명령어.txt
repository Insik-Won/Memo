■ 1일차 - vim 명령어

- 참조: https://vim.rtorr.com/lang/ko/

◎ 이동 명령
	- h, j, k, l: 좌, 상, 하, 우 이동
	-H, M, L: 현재화면 상단/중단/하단으로 점프
	- w, e: 다음 단어 시작/끝으로 점프
	- W, E: 다음 단어 시작/끝으로 점프(특수문자 포함)
	- b, ge: 이전 단어 시작/끝으로 점프
	- B, gE: 이전 단어 시작/끝으로 점프(특수문자 포함)
	- %: 현재 괄호 짝으로 점프
	- 0, $: 현재 행 시작/끝으로 점프
	- ^, g_: 현재 행 시작/끝으로 점프 (공백 미포함)
	- gg, G: 문서 첫줄/끝줄 시작으로 점프
	- gd, GD: 지역/전역 변수 선언 위치로 점프
	- fx, Fx: 다음/이전 검색 문자 x로 점프
	- tx, Tx: 다음/이전 검색 문자 x 앞/뒤로 점프
	- ; : 최근 f/t/F/T 명령어 실행
	- . : 최근 F/T/f/t 명령어 실행
	- {, }: 이전/다음 단락으로 점프
	- zz: 화면 위치 중간으로 고정
	- ^e, ^y: 한 줄 위/아래로 화면 조정
	- ^b, ^f: 한 화면 위/아래로 화면 조정
	- ^u, ^d: 반 화면 위/아래로 화면 조정
	
	- 팁: 이동 명령 앞에 숫자를 붙이면 그 수만큼 반복한다.
	
◎ 삽입 모드
	- i, I: 현재 커서 앞/행 시작에 삽입
	- a, A: 현재 커서 뒤/행 끝에 삽입
	- o, O: 현재 커서 행 아래/위에 새 행을 삽입
	- ea: 단어 끝 뒤에 삽입
	- ^h, ^w: 이전 문자/단어 제거
	- ^j: 한 줄 바꿈
	- ^t, ^d: 한 탭 들여쓰기/내어쓰기
	- ^p, ^n: 이전/다음 단어 탐색(자동완성)
	- ^rx: 레지스터 x의 내용 삽입
	- esc: 삽입모드 종료
	
◎ 편집 모드
	- r: 한 글자 바꾸기
	- R: esc 누를 떄까지 한 글자씩 바꾸기
	- J: 현재 행과 다음 행 연결
	- gJ: 현재 행과 다음 행 연결 (공백 무시)
	- gwip: 단락 리플로우
	- g~: 커서 이동으로 대-소문자 전환
	- gu/gU: 커서 이동으로 소문자/대문자 전환
	- cc: 한 행 전체 새로 쓰기
	- C, C$: 한 행 끝까지 새로 쓰기
	- ciw: 한 단어 전체 새로 쓰기
	- cw, ce: 한 단어 끝까지 새로 쓰기
	- s, S: 한 문자/행 새로 쓰기
	- u : 실행 취소
	- U: 최근 수정한 줄 복원
	- ^r: 다시 실행
	- .: 최근 명령어 반복
	
◎ 비쥬얼 모드
	- v: 비쥬얼 모드
	- V: 비쥬얼 라인
	- o: 선택 영역 반대쪽 끝으로 점프
	- ^v: 비쥬얼 블록
	- 0: 블록 반대쪽 모서리로 이동
	- aw: 단어 선택
	- ab, aB, at: 소괄호/중괄호/태그 구간 선택
	- ib, iB, it: 소괄호/중괄호/태그 내부 선택
		- b, B 대신 (, { 사용 가능
	- esc: 비쥬얼 모드 종료
	
◎ 비쥬얼 명령
	- >: 선택 행 들여쓰기
	- <: 선택 행 내어쓰기
	- y: 선택 구간 복사
	- d: 선택 구간 삭제
	- ~: 대소문자 반전
	- u,U: 선택구간 소문자/대소문자 전환
	
◎ 잘라내기와 붙여넣기
	- yy, dd: 한행 복사하기/잘라내기
		- 앞에 숫자 붙여서 행수 지정 가능
	- yw, dw: 다음 단어까지 복사/잘라내기
	- yiw, diw: 한 단어 복사/잘라내기
	- yaw, daw: 한 단어 복사/잘라내기 (공백 포함)
	- y$, d$: 한 행 끝까지 복사'잘라내기
	- p,P: 커서 뒤/앞에 붙여넣기
	- x: 한 문자 잘라내기
	
◎ 문단 모양
	- >>, <<: 한 탭 들여쓰기/내어쓰기
	- >%: 중/소괄호 구간 들여쓰기
	- >ib: 소괄호 내부 들여쓰기
	- >at: 태그 구간 들여쓰기
	- 3==: 3줄 자동정렬
	- =%: 중/소괄호 구간 자동정렬
	- =iB: 중괄호 내부 자동정렬
	- gg=G: 전체 버퍼 자동정렬
	- ]p: 붙여쓰고 현재 행 들여쓰기
	
◎ 검색과 바꾸기
	- /pattern, ?pattern: 패턴 검색 (순방향/역방향)
	- \vpattern: 특수 매직 패턴
	- n, N: 다음/이전 검색항목으로 점프
	- %s/old/new/g: 모든 old를 new로 바꾸기
	- %s/old/new/gc: 모든 old를 new로 확인하며 바꾸기

◎ 나가기
	- :w: 저장하기
	- :w !sudo tee %: 관리자로 저장하기
	- :wq, :x, ZZ: 저장하고 나가기
	- :q: 나가기 (변경확인)
	- :q!, ZQ: 나가기 (변경 무시)
	- :wqa: 모든 탭 저장하고 나가기
	
◎ 탭
	- :tabnew, :tabnew {page.words.file} - 새 탭에서 파일 열기
	- ^wT: 현재 분할 창을 새로운 탭으로 이동
	- gt, gT: 다음/이전 탭으로 이동
	- tabn[ext], tabp[revious]: 다음/이전 탭으로 이동
	- #gt, :tabm[ove] #: #번쨰 탭으로 이동
	- tabc[lose]: 현재 탭과 그 안의 창 닫기
	- tabo[nly]: 현재 탭 이외의 모든 탭 닫기
	- tabdo command: 모든 탭에서 command 실행하기
